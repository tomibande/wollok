# Assets necesarios para el juego

Este directorio debe contener las siguientes imágenes PNG (50x50 píxeles):

- `jugador.png` - Sprite del jugador
- `puerta_abierta.png` - Puerta abierta
- `puerta_cerrada.png` - Puerta cerrada
- `salida.png` - Puerta de salida
- `trampa.png` - Trampa
- `llave.png` - Llave
- `pocion.png` - Poción de vida

Puedes crear estas imágenes simples usando cualquier editor gráfico o generarlas online.
